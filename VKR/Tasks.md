## Задание 3 – «Методы работы с данными»

### Цель: подготовить данные как основу проекта, а не побочный продукт.

### Задание:

#### Разработайте план работы с данными:

#### Источники и сбор:

- Открытые датасеты: COCO для pretraining,
- Собственные данные: 1000 изображений с камер автобуса,
- Синтез: Blender-сцены для агрессивных жестов.

#### Предобработка:

- Удаление битых файлов, фильтрация по размытости (Laplacian variance < 100),
- Конвертация в RGB, нормализация к [0,1].

#### Разметка:

- Инструмент: CVAT,
- Политика: bounding box только по видимой части объекта; лица — размыти.

#### Разбиение:

- Train/Val/Test = 70/15/15,
- Стратификация по классам и условиям освещения,
- Групповое разбиение: все кадры одного автобуса – в одной выборке.

#### Аугментация:

- Color jitter, horizontal flip, motion blur (для имитации движения),
- Не использовать vertical flip (нефизично для салона).

#### Этика и закон:

- Анонимизация: blur лиц в реальном времени,
- Хранение: только эмбеддинги, без исходных кадров,
- Согласие: информирование пассажиров через таблички в салоне.
