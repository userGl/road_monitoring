## 2. Открытые датасеты

### 2.1. Детекция дефектов дорог

1. **RDD2022 (Road Damage Detection Dataset)**
   - Описание: Многонациональный датасет с 47 420+ изображений дефектов дорог из 6 стран
   - Типы дефектов: Ямы, трещины, выбоины, колейность
   - Ссылка: https://github.com/sekilab/RoadDamageDetector
   - Лицензия: CC BY 4.0

2. **Pavementscapes Dataset**
   - Описание: Крупномасштабный иерархический датасет для сегментации дефектов асфальтового покрытия
   - Размер: 4000+ изображений с высоким разрешением
   - Ссылка: https://github.com/ruirangerfan/pavement-damage-segmentation
   - Лицензия:

3. **CrackForest Dataset**
   - Описание: Датасет для сегментации трещин на дорожном покрытии
   - Размер: 118 изображений с масками трещин
   - Ссылка: https://github.com/cuilimeng/CrackForest-dataset
   - Лицензия: Открытый датасет
   - Примечание: 

4. **Cityscapes Dataset**
   - Описание: Датасет для калибровки сцены и сегментации городских сцен
   - Размер: 5000 изображений с плотной разметкой
   - Ссылка: https://www.cityscapes-dataset.com/
   - Лицензия: CC BY-NC-ND 4.0
   - Примечание:
